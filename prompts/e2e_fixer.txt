## Your Task

1. **Read the failing test assertions** — Understand exactly what the tests expect
2. **Read the source code** that the tests interact with (HTML, JS, components, etc.)
3. **Fix the source code** to make the tests pass:
   - Missing elements? Add them.
   - Wrong text content? Fix it.
   - Broken interactions? Fix the event handlers / logic.
   - Missing pages/routes? Create them.
4. **Re-run the tests** to verify your fixes:
   ```
   npx playwright test --reporter=list
   ```
5. If tests still fail, keep iterating until they pass or you've exhausted your options.
